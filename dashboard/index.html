All available API endpoints. API is bound to 10.13.13.1:8080 (WireGuard interface only).
GET /health Health check: status, total/verified/available proxy counts
GET /api/metrics JSON metrics: proxy counts, requests, connections, rotations
GET /metrics Prometheus-format metrics for monitoring (Grafana, etc.)
GET /api/proxies List active proxies: host, port, protocol, latency, country, success/fail count
POST /api/proxy/add Add proxy manually. Body: {"host":"...","port":...,"protocol":"http|socks5"}
POST /api/proxy/ban/{host:port} Ban a proxy (remove from active pool, add to banned list)
POST /api/proxy/unban/{host:port} Unban a proxy (restore from banned list to active pool)
GET /api/network-status Network info: LAN/WAN IP, UPnP status, WG port, peer count
GET /api/wg/peers WireGuard peers: name, address, public key, config availability
